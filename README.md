# Algoritmia
!Hola, este es un repositorio de mis trabajos de la asignatura de algoritmia cuales realicé tanto en clase como de manera personal¡

# ¿En qué consiste?
Se tratan de algoritmos hechos con pseudocodigo cuales resuelven problemas simples en diferentes contextos, estos pueden ser algoritmos secuenciales, selectivos, iterativos y con arreglos...
Mi objetivo es compartirlo para poder demostrar mi progreso.
